OOP Challenge

Car OO Challenge

  • Part 01 - Create a Class for Vehicle
  • Part 02 - Create a Class for Car that extends Vehicle and adds a property
  • Part 03 - Create a Motorcycle Class that extends Vehicle and should have numWheels property and a revEngine mehtd
  • Part 04 - Create a class for a Garage. It should have a property called vehicles which will store an array of vehicles, and a property called capacity which is a number indicating how many vehicles will fit in the garage. When you create a garage, vehicles will always be empty; you only need to provide the capacity.
